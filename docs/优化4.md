1. 品牌与顶部全局定义 (Header)
系统更名：

主标题由 CryptoRadar 变更为 monitor。

子标题由 量化交易与风控指挥中心 变更为 加密货币信号监测系统。

视觉规范：继续保持苹果极简科技风，确保汉化后的字体在深色模式下清晰、整洁。

2. 仪表盘页优化 (Dashboard)
2.1 账户资产明细：

新增字段：在原有基础上，必须同时展示“账户总资金”、“可用保证金”及“总计未实现盈亏”三个核心指标。

2.2 持仓卡片增强：

动态展示：在当前的持仓卡片中，除了仓位大小和方向，需实时显示该单的“未实现盈亏”。

2.3 实时价格看板：

位置：在“账户与持仓”卡片正上方。

逻辑：根据“系统设置”页中已选中的监控币种，实时滚动显示其币价（如 BTCUSDT, ETHUSDT 等）。

3. 信号雷达页精细化 (Signal Radar)
3.1 列表字段调整：

新增展现：表头增加“入场价”字段。

隐藏展示：将“影线占比”、“EMA 距离”、“ATR 波动率”从列表首屏隐藏（保留在数据模型中供详情查看）。

3.2 操作与查询功能：

维度查询：增加“时间范围查询条件”（如：最近 1 小时、24 小时、自定义）。

数据管理：支持“批量删除”已选信号，以及“单笔手动删除”功能。

3.3 详情弹窗机制：

交互：每行数据末尾新增“详情”按钮。

内容：点击后弹出半透明毛玻璃卡片，完整展示该信号的所有底层计算字段（包含影线占比等隐藏指标）。

4. 系统设置页功能对齐 (Settings)
4.1 告警多通道支持：

企微集成：在飞书通知下方，同步增加“企业微信推送”开关及“企微 Webhook 密钥”配置项。

4.2 系统总开关逻辑拆分：

全局推送开关：默认开启，可编辑。作为通知的总闸，开启后才执行飞书/企微的逻辑判断。

自动下单开关：默认关闭，严禁编辑且必须置灰显示，确保系统仅作为信号监测工具。

4.3 币种配置交互优化：

动态追加：在预设币种（BTC, ETH, SOL, BNB）后方增加一个“+ 添加”按钮。

逻辑：点击后实时向列表追加一个新币种对，新币种默认不选中。
