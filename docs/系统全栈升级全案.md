- ## 核心算法：动态加权评分引擎 (Dynamic Scoring)

  系统不再使用固定的打分逻辑，而是允许通过配置界面实时调整以下三个维度的权重，总分为 100 分：

  - **形态完美分 ($S_{shape}$)**：影线与实体的黄金比例计算。
  - **趋势回归分 ($S_{trend}$)**：价格相对于 EMA60 的位置。
  - **波动爆发分 ($S_{vol}$)**：当前波幅对比 ATR(14) 的健康度。

  **打分公式：**

  $$Score = (S_{shape} \times W_{shape}) + (S_{trend} \times W_{trend}) + (S_{vol} \times W_{vol})$$

  *(其中权重 $W$ 之和为 1，用户可在设置页自由分配比例)*

  ## 📊 前端模块：苹果极简指挥中心

  ### A. 仪表盘 (Dashboard) - 实时监控

  - **系统健康状态条**：展示 WebSocket 连接状态（脉冲呼吸灯）、API 延迟、以及系统已持续运行时间 (Uptime)。
  - **性能监控进度条**：实时显示**币安 API 权重消耗百分比**。这对于防止因高频请求被封禁 IP 至关重要，尤其是在多币种监控模式下。
  - **资产快照**：展示实时余额与当前持仓卡片，采用 Apple Card 风格。

  ### B. 信号雷达 (Signals) - 筛选与复盘

  - **高分展示**：列表显眼位置标注信号得分，并根据分数级别显示不同色彩深度。
  - **条件查询**：
    - **过滤器**：支持按币种、多空、时间范围、**最低分数线**过滤。
    - **统计小结**：今日/本周信号数、预期盈亏比展示。
  - **批量管理**：支持手动勾选删除，以及“一键清理 7 天前记录”。

  ### C. 系统配置 (Settings) - 分级管理

  - **交易所集成**：API Key 密文管理、**多币种勾选**、时间周期选择。
  - **通知中心**：推送总开关、飞书/企微 Webhook、**发送测试消息**按钮。
  - **策略权重配置**：**新增三个滑动条**，用于动态分配 Shape、Trend、Vol 的权重占比。
  - **风控参数**：单笔风险比例、最大杠杆。

  ------

  ## 🛠️ 后端架构：安全与韧性设计

  - **配置热加载**：设置保存后即刻生效，无需重启主引擎。
  - **敏感数据保护**：API Secret 在数据库中进行**加密存储**，防止明文泄露。
  - **遥测系统**：后端定期计算 API 权重消耗并推送到前端，确保 24/7 运行的稳定性。
  - **强制告警**：即使关闭推送，若发生网络持续断开或 API 失效等重大故障，系统将强制发送通知。
